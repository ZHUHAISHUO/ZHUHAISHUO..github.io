Angular4.0搭建新项目


前言：最近公司新项目或许会用到Angular4，所以事先先了解一下如何搭建一个基本项目以及创建项目所需要的环境配置等。以下是Angular4.0基本项目的搭建过程。



一、安装node.js

1. 高版本node包含npm，不用单独进行安装，如果node版本过低需要再单独安装npm
2. 安装好node.js之后检测node版本与npm版本,看是否安装成功

node -v

npm -v


(二)安装angular-cli工具

1.输入指令，进行安装。
npm install -g @angular/cli

2.等待安装完成之后检测是否安装成功
ng -v


(三)开始创基本建项目

1.输入指令
ng new product
(product是项目名称)。
2.打开新创建的项目，可以看到一个基本项目的目录结构了。

(四)Angular4引入第三方库步骤描述

1.引入jQuery：命令行中cd到新建项目文件夹下输入指令
npm install jQuery –save

2.引入bootstrap：同样的指令
npm install bootstrap –save

3.安装完第三方插件后，打开项目目录的node_module文件夹会找到刚才安装的jQuery与bootstrap。

4.需要注意的是Angular4是基于ES6的typescript语法写的，所以我们引入了jQuery和bootstrap后并不能直接使用，我们还需要安装typescript来识别它们，输入指令：
npm install @types/bootstrap –save-dev

npm install @types/bootstrap –save-dev


(五)使用Angular-cli工具生成组件/模块

1.如果说我们需要创建一个头部导航的组件，可以使用指令
ng -g component head
(head是组件名称)。
2.基本的组件就创建成功后，打开项目src/app/文件目录会发现多了个文件夹head，然后就可以对文件夹中的html、css文件进行编译了。
3.angular有依赖注入的特性，所以在安装第三方插件和head，foot模块时需要在后缀名为.ts的文件中配置。
