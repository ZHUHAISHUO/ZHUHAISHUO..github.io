Vue2.0搭建新项目


1. 安装 node.js

npm -v 检测是否安装成功

2. 安装淘宝镜像 cnpm

npm install -g cnpm –registry=http://registry.npm.taobao.org


3. 安装构建工具 Vue-cli

npm install -g vue-cli

4. 创建项目

vue init webpack myVue

使用该指令意思是初始化项目，其中的webpack是构建工具，也就是整个项目是基于webpack的。

这里需要注意的是要进入存放项目的目录下安装，否则会安装到默认目录下。

安装成功后会提示输入一些信息，直接回车进入下一步

5. 安装项目所需依赖

cnpm install

项目安装成功后需要安装依赖才能运行，首先cd到项目目录下，运行 cnpm install

安装完成之后会看到项目目录下多了一个node_modules,这就是刚才安装的依赖模块。

6. 运行项目

npm run dev

在项目目录下运行指令，这条指令会使用热加载的方式运行我们的应用，热加载可以让我们在修改完代码后不用手动刷新浏览器就能实时看到修改后的效果。

运行成功后浏览器会自动打开localhost:8080，这时就可以看到项目运行起来的页面了。
