vue2.0中各文件之间的引用加载关系


1.如下图：main.js 是我们的入口文件，主要作用是初始化vue实例并使用需要的插件








        import Vue from ‘vue’
        import App from ‘./App’
        / eslint-disable no-new /
        new Vue({
          el: ‘#app’,
          template: ‘‘,
          components: { App }
        })



2.App.vue是我们的主组件,所有页面都是在App.vue里边自定义的组建中加载的，所有组件都是App.vue的子组件



        template
        div id=”app”
            hello /hello
        div
        template
        script
            import Hello from ‘./components/Hello’
            export default {
            name: ‘app’,
            components: {
            Hello
            }
        }
        script>



1.index.html文件入口
2.src放置组件和入口文件
3.node_module为所有依赖的模块
4.config中配置了路径端口值等
5.build中配置了webpack的基本配置、开发环境、生产环境配置等
