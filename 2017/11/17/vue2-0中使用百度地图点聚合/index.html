vue2.0中使用百度地图点聚合

此文讲述在vue中使用百度地图点集合功能，在使用点聚合功能时需要以下几项配置：

一.npm安装Echarts： npm install echarts – save 安装成功后Echarts会在node_modules目录下

二.引入Echarts：(三种方式)


1可直接使用线上cdn地址，在index.html中引用一个script标签:

<script src=”https://cdn.bootcss.com/echarts/3.8.4/echarts.min.js"></script&gt;

2在main.js中全局引用：

import Echarts from ‘Echarts’; // 引入Echarts
Vue.use(Echarts); //  使用Echarts

3按需引入：

require(‘echarts/lib/echarts’);// 引入 ECharts 主模块
require(‘echarts/lib/chart/bar’);// 引入柱状图
require(‘echarts/lib/component/tooltip’);// 引入提示框组件
require(‘echarts/lib/component/title’);// 引入标题组件

三.使用Echarts：引入成功后，在template中定义一个div，加上ID ，并设置高度，若高度不设置，地图将不会显示。

1初始化：

let myChart = echarts.init(document.getElementById(“clear_work”)); //  初始化地图，这里clear_work为id

2插入opation数据：

Chart.setOption(this.clear_work_obj); //  插入数据，这里clear_work_obj是opation，只是自己取得名字方便区分
window.addEventListener(‘resize’, function () { //  Canvas宽度自适应
   myChart.resize(); // 使用resize方法，动态设置容器的宽度，用于自适应图表
});
