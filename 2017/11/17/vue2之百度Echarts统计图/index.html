vue2.0中使用百度Echarts

发表于 2017-11-17


前言：在vue中使用百度Echartst图表：分为普通图表和地图图表两种。描述Echarts的引入途径及使用方法。


一、npm安装Echarts：安装成功后Echarts会在node_modules目录下

  1.  npm install echarts -- save

二、引入Echarts：(三种方式)

  1. 可直接使用线上cdn地址，在index.html中引用script标签:
     https://cdn.bootcss.com/echarts/3.8.4/echarts.min.js
  2. 在main.js中全局引用：
    import Echarts from 'Echarts'; // 引入Echarts
Vue.use(Echarts); //  使用Echarts
  3. 按需引入：
    require('echarts/lib/echarts');// 引入 ECharts 主模块
require('echarts/lib/chart/bar');// 引入柱状图
require('echarts/lib/component/tooltip');// 引入提示框组件
require('echarts/lib/component/title');// 引入标题组件

三、使用Echarts

  1. 初始化：引入成功后，在template中定义一个div，加上ID ，并设置高度，若高度不设置，地图将不会显示。
    let myChart = echarts.init(document.getElementById("clear_work")); //  初始化地图，clear_work为id
  2. 插入opation数据：
    Chart.setOption(this.clear_work_obj); //  插入数据，这里clear_work_obj是opation，只是自己取得名字方便区分
    window.addEventListener(‘resize’, function () { //  Canvas宽度自适应
    myChart.resize(); // 使用resize方法，动态设置容器的宽度，用于自适应图表
    });

    四、完成上述步骤后，一个完整的Echarts图表就生成了。
