Atom编辑器不能安装插件问题


前言：发现一款精致的编辑器Atom，但是在使用vue时，XXX.vue文件并不支持高亮显示。编辑器内部的install a Package和apm install都不支持安装，最终找到一个通过github解决的方法。


一、问题：

1. 使用Install Packages安装失败，提示请求超时。可能是在大天朝访问国外服务器导致。
2. 使用npm分支下的apm：apm install atom-vue，同样出现错误。

一、解决方法：

1、在github上搜索自己想要安装的插件：如 atom-vue atom。并复制仓库地址。
2、打开atom安装目录，找到packages文件，在此文件下 git bush here，打开git窗口。
3、将复制的仓库地址克隆到本地。
4、将复制的仓库地址克隆到本地。就会发现packages文件夹下多了一个atom-vue文件夹。重启atom，ok了。
